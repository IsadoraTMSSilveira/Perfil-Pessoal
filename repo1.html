REPOSITÓRIO: Construção de Curriculum
Descrição:

Conhecer como funciona o processo de comunicação entre cliente e servidor para a carga e processamento de um site. Explicar cada passo executado por estes componentes para que o usuário possa visualizar o site no navegador da Internet. Conhecer a estrutura básica das páginas Web, a linguagem HTML, onde irá passar pelo histórico da linguagem HTML, a sintaxe para a criação de páginas e os principais elementos. Sabendo assim, descrever o processo de carga e processamento de um site, entender a interligação de arquivos que formam uma página Web e montar a estrutura de um arquivo HTML para criar páginas da Web completas
